# NS White Crane White Belt

A TypeScript monorepo for coding katas

## Structure

- **`packages/katas/`** - Coding exercises
- **`packages/shared/`** - Shared libraries
- **`packages/config/`** - Config packages
- **`packages/templates/`** - Project templates

## Quick Start

```bash
# Install dependencies
npm install

# Build all packages
npm run build
```
